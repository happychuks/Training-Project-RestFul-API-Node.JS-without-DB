1. npm init -y (to install package.json)
2. npm install --save express (to install package-lock.json and node_modules)
3. include "type": "module" in the package.json file to enable the newest Import function
4. npm install --save-dev nodemon (this is dependency that enables autosave changes made in packages of our development, so you don't have to stop ur server and run again. NOTE: This dependency is only applicable to the current project. you will have to install anytime its needed in a new project)
5. npm start (to run the program)
6. to make a POST request on postman, switch to post, goto body, select raw, change text to Json
7. npm install uuid (to install package to create unique ID) https://www.npmjs.com/package/uuid
8. to update a user data. use PATCH to make partial update while PUT is to completely overwrite your user data
9. CTRL+C to stop npm from running.
